# Project Summary

**Project title:** Horse Colic Diagnosis with Machine Learning

This repository is a portfolio-ready adaptation of a sophomore spring Machine Learning course final project.  
The task is to diagnose horse colic cases from structured clinical indicators using classical machine learning methods.

## Methods
- Data cleaning
- Feature naming and preprocessing
- Candidate categorical feature detection
- Standardization
- PCA-based feature analysis
- Logistic Regression
- Random Forest
- AdaBoost
- Hyperparameter tuning with GridSearchCV / RandomizedSearchCV

## Key takeaway
Among the compared models, Random Forest delivered the strongest overall performance in the original coursework, while AdaBoost was useful for higher-recall screening and Logistic Regression provided the clearest interpretability.
